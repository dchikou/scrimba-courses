How To Play

Start Game: The first player is chosen at random, with either side having a 50/50 chance of going first.

Roll Dice: The game is set and now you're off! Each roll of the dice will add the value of the roll (1-6) to your total in a race for 30 points.

Double Or Nothing: Feeling lucky? Take a chance! Earn double points for even numbers; LOSE double points for odds!

Reset Game: One way or another the game is done and a winner declared. Want to play again? This button resets the game back to the start.
